# TheDeductionist

A premium, AI-powered detective deduction game. 

## Features
- **AI Narrative Engine**: Leverages Gemini 2.5 Flash-Lite to rephrase clues into atmospheric noir prose.
- **Dynamic Dossier**: Detailed suspects, weapons, and locations with rich backstories and unique trace evidence.
- **Responsive Logic**: Fully mobile-responsive UI with a sleek cyberpunk noir aesthetic.
- **Customizable Difficulty**: Multiple grid sizes and case complexities.

## Tech Stack
- React + TypeScript + Vite
- Lucide-React Icons
- Framer Motion
- Gemini API (v1beta)

## Getting Started
1. Clone the repo.
2. Install dependencies: `npm install`
3. Start the dev server: `npm run dev`
4. Enter your Gemini API key in the in-game settings to enable the AI Narrative Engine.
